<h2><a href="https://leetcode.com/problems/43um/?envType=problem-list-v2&envId=w5lnneo1">15. 3Sum</a></h2><h3>Medium</h3><hr><p>Given an array <code>nums</code> of <code>n</code> integers, return <em>all the triplets</em> <code>[nums[i], nums[j], nums[k]]</code> such that:</p>

<ul>
	<li><code>i != j</code>, <code>i != k</code>, <code>j != k</code></li>
	<li><code>nums[i] + nums[j] + nums[k] == 0</code></li>
</ul>

Notice that the solution set must not contain duplicate triplets.

<p>&nbsp;</p>
<p><strong class="example">Example 1:</strong></p>

<pre>
<strong>Input:</strong> nums = [-1,0,1,2,-1,-4]
<strong>Output:</strong> [[-1,-1,2],[-1,0,1]]
</pre>

<p><strong class="example">Example 2:</strong></p>

<pre>
<strong>Input:</strong> nums = [0,1,1]
<strong>Output:</strong> []
</pre>

<p>&nbsp;</p>
<p><strong>Constraints:</strong></p>

<ul>
	<li><code>3 &lt;= nums.length &lt;= 3000</code></li>
	<li><code>-10<sup>5</sup> &lt;= nums[i] &lt;= 10<sup>5</sup></code></li>
</ul>
